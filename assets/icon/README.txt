Uygulama ikonu (logo) ve açılış ekranı (splash) bu klasöre konur.

1. Logo dosyanızı buraya "app_icon.png" adıyla kopyalayın.
   Önerilen boyut: 1024x1024 piksel, PNG, şeffaf arka plan kullanılabilir.

2. Terminalde proje kökünden sırayla:
   dart run flutter_launcher_icons
   dart run flutter_native_splash:create

3. Uygulamayı tam yeniden derleyin (flutter run).

Böylece hem ana ekran ikonu hem de açılış ekranı (splash) aynı logoyu kullanır.
